# Unity3D 2022 LTS 基础能力问答 - LLM答题标准提示词

## 提示词模板

```
你是一个Unity3D 2022 LTS的技术专家。请回答以下选择题。

【输出格式要求】严格遵守以下格式，不要有任何额外解释或文字：
1. 每行一题，格式为：Q编号. 答案字母
2. 单选题只输出一个字母（A/B/C/D）
3. 多选题输出所有正确选项字母，按字母顺序排列，不要空格（如ABC、ABCD）
4. 判断题输出A（正确）或B（错误）
5. 不要输出任何解释、原因、分析或空行
6. 编号使用三位数格式（如Q001, Q002...Q1000）

【输出示例】
Q001. C
Q002. B
Q003. D
Q004. ABCD
Q005. A

---

以下是题目，请开始作答：

{questions_content}
```

## 使用说明

1. 将上述提示词中的 `{questions_content}` 替换为 `questions.md` 中的题目内容
2. 如果题目超过模型上下文窗口限制，可分批提交（如每次100题）
3. 分批时提示词不变，只替换题目部分
4. 收集所有批次的输出后合并为一个文件用于评分

## 分批提示词模板（适用于上下文窗口较小的模型）

```
你是一个Unity3D 2022 LTS的技术专家。请回答以下选择题（第{batch_start}-{batch_end}题，共{total}题中的一部分）。

【输出格式要求】严格遵守以下格式：
- 每行一题：Q编号. 答案字母
- 单选题: A/B/C/D
- 多选题: 按字母顺序排列（如ABC）
- 判断题: A（正确）或B（错误）
- 不要输出任何解释

{batch_questions}
```

## LLM答案输出示例文件格式（llm_answer.txt）

```
Q001. C
Q002. B
Q003. D
Q004. A
Q005. A
Q006. A
Q007. A
Q008. A
Q009. D
Q010. B
...
Q1000. A
```

## 评分方式

使用 `Unity3d/score_answers.py` 脚本评分：

```bash
python3 Unity3d/score_answers.py --answer answers.md --llm llm_answer.txt
```

输出包含：
- 总分（百分比）
- 各模块得分
- 各维度得分
- 各难度等级得分
- 各题型得分
- 错误题目清单
